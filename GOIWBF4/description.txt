0.6.0版本：
降低了锤子给予玩家的整体速度大小（降低加速度基础大小）
新增时间回溯功能，可以回溯几秒前的位置，需要在控件中设置时间回溯按键
新增模组配置，在模组配置-锤子配置里，你可以：
	修改锤子给予玩家X/Y方向上的速度大小的倍率
	修改时间回溯的时间间隔和记录个数
加入锤子碰撞时的声音（采用挖掘的声音）
本版本作为最终版本开源并且不再更新，望周知
Version 0.6.0: 
Reduced the overall speed given to the player by the hammer (reducing the basic acceleration) 
New time backtracking function added, which can backtrack the position a few seconds ago. The Back In Time Button needs to be set in the Controls
Add Mod configuration. In Mod Configuration-Hammer Config, you can: 
	Modify the multiplier of the speed given to players in the X/Y direction by the hammer 
	Modify the time interval and number of records for time backtracking
Add the sound of hammer collision (using the sound of digging)
This version is open source as the final version and will no longer be updated. Please be informed
